# COMPUTER NETWORKING FUNDAMENTALS

## A Brief Overview of Computer Networks

## What is a Computer Network?

A computer network is a group of devices connected to each other in order to exchange information. These devices can include computers, smartphones, printers, servers, and other equipment.

## INTERNET
is the largest example of a network that exists today.
Networks allow:
*File sharing;
*Internet access;
*Communication between devices;
*Resource sharing, such as printers and servers.

## TCP/IP 
Is the suite of protocols responsible for communication between devices on a network and on the Internet.

## TCP (Transmission Control Protocol)
Ensures that data is sent and received correctly, without loss and in the proper order.
Example:
When you send a message through WhatsApp, TCP helps ensure that it arrives complete and intact to the recipient.

## IP (Internet Protocol)
Is responsible for identifying each device on a network through an IP address.
Example:
Just as a house has an address to receive mail, a computer has an IP address to receive data.

## DNS 
Stands for Domain Name System.
Its function is to translate website names into IP addresses.
Example:
When we type: www.google.com
DNS finds the corresponding IP address so that the browser can access the website.
Without DNS, it would be necessary to memorize IP numbers to access websites on the Internet.

## DHCP
Stands for Dynamic Host Configuration Protocol.
Its function is to automatically provide network settings to devices.
These settings include:
*IP address;
*Subnet mask;
*Gateway;
*DNS server.
Example:
When you connect your smartphone to your home Wi-Fi network, it usually receives all these settings automatically through DHCP.

#3 HTTP
Stands for HyperText Transfer Protocol.
It is the protocol used for communication between web browsers and web servers.
When you access a website, your browser uses HTTP to request information from the server.

### Characteristics:
Does not provide encryption;
Data can be viewed during transmission.

## HTTPS
Stands for HyperText Transfer Protocol Secure.
It is a more secure version of HTTP.
It uses encryption to protect data transmitted between the browser and the server.
Advantages:
*Greater security;
*Protection of passwords and personal data;
*Increased user trust.
Example:
Banking websites, online stores, and social media platforms use HTTPS to protect users' information.

##Conclusion

The TCP/IP, DNS, DHCP, HTTP, and HTTPS protocols are fundamental to the operation of the Internet and computer networks. Understanding their basic concepts is an important step for anyone beginning their studies in Information Security.

# FUNDAMENTOS DE REDES DE COMPOUTADORES

## Um breve resumo sobre rede de computadores

## O que é uma rede de Computadores? 
É um conjunto de dispositivos conectados entre si para trocar informações. Esses dispositivos podem ser computadores, celulares, impressoras, servidores e outros equipamentos.

## INTERNET 
É o maior exemplo de rede existente atualmente.
As redes permitem:
*Compartilhamento de arquivos;
*Acesso à internet;
*Comunicação entre dispositivos;
*Compartilhamento de recursos, como impressoras e servidores.

## TCP/IP 
É o conjunto de protocolos responsável pela comunicação entre dispositivos em uma rede e também na internet.
TCP (Transmission Control Protocol)
O TCP garante que os dados sejam enviados e recebidos corretamente, sem perdas e na ordem certa.
Exemplo:
Quando você envia uma mensagem pelo WhatsApp, o TCP ajuda a garantir que ela chegue completa ao destinatário.

## IP (Internet Protocol) 
É responsável por identificar cada dispositivo na rede através de um endereço IP.
Exemplo:
Assim como uma casa possui um endereço para receber correspondências, um computador possui um endereço IP para receber dados.

## DNS 
Significa Domain Name System.
Sua função é transformar nomes de sites em endereços IP.
Exemplo:
Quando digitamos: www.google.com
o DNS encontra o endereço IP correspondente para que o navegador consiga acessar o site.
Sem o DNS, seria necessário decorar números de IP para acessar páginas da internet.

# DHCP 
Significa Dynamic Host Configuration Protocol.
Sua função é fornecer automaticamente configurações de rede para os dispositivos.
Entre elas:
*Endereço IP;
*Máscara de rede;
*Gateway;
*Servidor DNS.
Exemplo:
Quando você conecta seu celular ao Wi-Fi de casa, normalmente ele recebe todas essas configurações automaticamente através do DHCP.

## HTTP 
Significa HyperText Transfer Protocol.
É o protocolo utilizado para a comunicação entre navegadores e servidores web.
Quando você acessa um site, seu navegador utiliza HTTP para solicitar as informações ao servidor.

### Características:
Não possui criptografia;
Os dados podem ser visualizados durante a transmissão.


## HTTPS 
Significa HyperText Transfer Protocol Secure.
É uma versão mais segura do HTTP.
Ele utiliza criptografia para proteger os dados transmitidos entre o navegador e o servidor.

Vantagens:
*Maior segurança;
*Proteção de senhas e dados pessoais;
*Mais confiança para os usuários.

Exemplo:
Sites de bancos, lojas virtuais e redes sociais utilizam HTTPS para proteger as informações dos usuários.

## Conclusão
Os protocolos TCP/IP, DNS, DHCP, HTTP e HTTPS são fundamentais para o funcionamento da internet e das redes de computadores. Conhecer seus conceitos básicos é um passo importante para quem está iniciando os estudos em Segurança da Informação.
